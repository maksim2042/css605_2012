    def getFOV(self,x,y,radius):

        rows = self.env[self.wrap(x-radius):self.wrap(x+radius)]
        for row in rows:
            fov.append(row[self.wrap(y-radius):self.wrap(y+radius)])

So what is the problem ?

I have a rabbit sitting at 8,1 on a dim 10 grid with a vision radius of 3. 
The vision of the rabbit is going to wrap around the torus so the lower index will be greater then higher index. 
Python is kind enough not to blow up on you, but it will return an empty list [].

A simplified example of the problem:

>>> a = [ 1,2,3,4,5,6,7,8,8]
>>> a [5:3]
[]


Suggested fix:

    def getFOV(self,x,y,radius):
        fov=[]
        x_range = set([self.wrap(i) for i in range(x - radius, x + radius +1)])
        y_range = set([self.wrap(i) for i in range(y - radius, y + radius +1)])


        for x in x_range:
            row = []
            for y in y_range:
                row.append(self.env[x][y])
            fov.append(row)

        return fov
